# AI Disclosure and Portfolio Context

This repository is part of the **Spring 2026 AI Research Prototype Portfolio (S26 AIRP)**, a spring 2026 portfolio of **AI-assisted research software prototypes** created by Ryan Kamp, a graduate student in computer science at the University of Cincinnati.

## Project purpose

This repository belongs to **S26 AIRP**, an exploratory portfolio on **AI-assisted research software prototyping** and **LLM-assisted scientific software development**. The portfolio used scientific and research-themed topics as testbeds for learning how modern AI tools can support software prototyping, Streamlit application development, technical documentation, LaTeX report generation, and research-style workflow design.

The primary value of this repository is technical, methodological, and educational. It is presented as a software-engineering, workflow-design, and AI-methodology artifact, not as validated scientific research.

## Role of AI assistance

This repository was developed with substantial AI assistance. AI tools were used throughout the workflow for activities such as software prototyping, code generation, debugging support, documentation, report drafting, literature-review workflow experimentation, LaTeX formatting, prompt engineering, and context engineering.

The primary model used in this portfolio workflow was Claude Opus 4.6, as recorded by the project owner. The repository should therefore be understood as an AI-assisted artifact rather than as a conventionally authored research project.
